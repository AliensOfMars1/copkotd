# Images Directory

Place your church images here. All images should be properly licensed for use.

## Recommended Image Sources:
- Unsplash (https://unsplash.com) - Free high-quality photos
- Pexels (https://pexels.com) - Free stock photos
- Pixabay (https://pixabay.com) - Free images

## Recommended Image Sizes:
- Hero Background: 1920x1080px
- Ministry Cards: 600x600px
- Welcome Section: 800x600px
- Sermon Thumbnail: 1280x720px

## Image Files to Add:
1. hero-bg.jpg - Main hero background image
2. church-community.jpg - Welcome section image
3. youth-ministry.jpg - Youth ministry card
4. mens-ministry.jpg - Men's ministry card
5. womens-ministry.jpg - Women's ministry card
6. childrens-ministry.jpg - Children's ministry card
7. outreach-ministry.jpg - Outreach ministry card
8. sermon-thumbnail.jpg - Latest sermon preview
9. church-logo.png - Church logo (replace the ✝ emoji)

## Usage:
After adding images, update the src attributes in index.html 
and background-image properties in style.css
